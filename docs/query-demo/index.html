Query otlp2parquet Samples in Your Browser

This interactive demo loads deterministic Parquet fixtures that were generated by running otlp2parquet against the repository test data. Everything runs locally via DuckDB-Wasm – no data leaves your browser.

1. Sample datasets

The following Parquet files are registered in DuckDB when the page loads:

  • logs.parquet OTLP log records with severity, trace context, and attributes.
  • metrics_gauge.parquet Gauge data points for the cpu.usage metric.
  • traces.parquet Trace spans with events, links, and status metadata.
