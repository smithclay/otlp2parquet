OpenTelemetry semantic conventions for resource attributes.

These are well-known attribute keys defined by the OpenTelemetry specification for common resource properties. We extract some of these to dedicated columns for better query performance and ClickHouse compatibility.

Reference: https://opentelemetry.io/docs/specs/semconv/resource/
